# Instrucción: evaluar cumplimiento financiero / regulatorio

El documento ya fue anonimizado con Presidio si contiene datos personales.
Evaluar el documento contra la regulación indicada en el skill de cumplimiento activo.

## Tu tarea

1. Identificar el tipo de documento y la regulación aplicable (provista por el skill)
2. Aplicar el checklist correspondiente del skill
3. Identificar cada obligación incumplida con norma específica
4. Clasificar riesgos: Alto · Medio · Bajo
5. Proponer corrección concreta para cada hallazgo
6. Determinar si requiere revisión de experto antes de continuar operando

## Formato de respuesta obligatorio

```
## TIPO DE DOCUMENTO
[identificado automáticamente]

## REGULACIÓN APLICADA
[nombre de la norma principal del skill]

## CHECKLIST
| Obligación | Norma | Estado | Riesgo |
|------------|-------|--------|--------|
| [item] | Art. XX | ✓ / ✗ / Parcial | - |
...

## HALLAZGOS
### HALLAZGO N
- Descripción: ...
- Norma: [Art. XX / NCG XXX]
- Riesgo: Alto / Medio / Bajo
- Corrección: [texto o acción concreta]

## ¿REQUIERE REVISIÓN EXPERTA?
Sí / No / Evaluar — [1 línea de justificación]

## RESUMEN EJECUTIVO
[estado general · riesgo principal · acción prioritaria]
```

## Restricciones

* Citar siempre la norma exacta (artículo, número de circular, resolución)
* No emitir opinión legal o contable vinculante
* Si el documento está incompleto para evaluar → indicarlo antes de continuar
* Separar hechos de interpretación
