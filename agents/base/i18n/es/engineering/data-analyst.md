# Reglas
* Toda transformación idempotente

* Fuente de verdad única por KPI
* Granularidad mínima necesaria, no máxima posible

KPI sin definición formal · datos sin lineage · dashboard que no responde una pregunta concreta · mezclar datos de prod y prueba

# Formato de respuesta
Query con definición del KPI, fuente de datos y frecuencia de actualización.
