# Regulación aplicable: Ley 20.584 — Derechos y Deberes del Paciente (Chile)

> Este skill provee el conocimiento de dominio que usa el agente `compliance-officer`.
> Usar junto con `cumplimiento-ley-21719.md` para datos de salud (son datos sensibles).

## Derechos fundamentales del paciente

### Derecho a la información (Art. 8-12)
```
□ Diagnóstico informado en términos comprensibles
□ Alternativas de tratamiento presentadas
□ Riesgos y beneficios de cada alternativa explicados
□ Pronóstico comunicado con honestidad
□ Derecho a no ser informado (si el paciente lo solicita)
□ Información entregada con tiempo suficiente para decidir
```

### Consentimiento informado (Art. 14-16)
```
□ Consentimiento por escrito para procedimientos invasivos
□ Firmado antes del procedimiento, no después
□ Capacidad del paciente para consentir verificada
□ Representante legal si el paciente no puede consentir
□ Revocación del consentimiento: derecho en cualquier momento
□ Excepciones: urgencia vital o riesgo para terceros
```

### Confidencialidad y ficha clínica (Art. 12-13)
```
□ Ficha clínica es propiedad del prestador, pero el paciente tiene derecho de acceso
□ No compartir información clínica sin autorización del paciente
□ Excepciones: notificación de enfermedades de reporte obligatorio (SEREMI)
□ Plazo de conservación ficha clínica: mínimo 15 años
□ Datos de salud = datos sensibles bajo Ley 21.719: requieren consentimiento explícito
```

### Trato digno (Art. 5-7)
```
□ No discriminación por condición económica, origen, enfermedad
□ Privacidad durante exámenes y procedimientos
□ Acompañante permitido salvo razón clínica justificada
□ Atención en idioma comprensible (intérprete si se requiere)
```

## Checklist por documento

### Consentimiento informado
```
□ Identificación del paciente y del prestador
□ Procedimiento descrito en lenguaje claro
□ Riesgos específicos mencionados (no genéricos)
□ Alternativas ofrecidas y documentadas
□ Firma del paciente (o representante) y fecha
□ Firma del profesional que informa
□ Copia entregada al paciente
```

### Ficha clínica / epicrisis
```
□ Identificación completa del paciente (anonimizar antes de analizar)
□ Diagnóstico con código CIE-10
□ Evolución clínica registrada por fecha
□ Medicamentos con dosis, vía y frecuencia
□ Consentimientos informados adjuntos
□ Alta médica con indicaciones escritas
```

## Sanciones de referencia

| Infracción | Consecuencia |
|------------|-------------|
| Procedimiento sin consentimiento | Denuncia ante Superintendencia de Salud |
| Filtración de ficha clínica | Ley 21.719 + Ley 20.584 |
| Negación de acceso a ficha | Recurso de protección |
